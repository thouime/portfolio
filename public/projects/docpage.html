Why Python?

Python is a wonderful language to start learning programming with because it's simple. The language is not complicated, it's straight to the point without excess, and syntax is easy to read for beginners to programming.

A common problem when faced with the task of learning a programming language is weaving in and out of using and learning the syntax. This becomes a problem of causing confusion with learning a language that should really have the priority of learning the concepts more than focusing too much on the syntax.

Below is a code snippet from Java that prints out a name, Frank, to the console.

public class PrintName { public static void main(String[] args) { // Set the name variable to frank String name = "Frank"; // Print out the name System.out.println(name); } }

In comparison, below is some code from python and performs the same function that's easier to read and understand.

# variable that contains a name name = "Frank" # Print out the name print(name)
Variables

Variables are containers that hold information that can be changed or used throughout your code.

Variables can hold different types of information that include:

Variable Types:

  • Integers: Numbers.
  • Strings: A series of characters making up a word or multiple words.
  • Booleans: A type of variable that can be either set to True or False.
  • Lists: A list of multiple variables grouped together.
  • Tuple: A group of items that are ordered and cannot be changed later.
  • Dictionary: A group of items that are paired together each with a key and a vlue.

Seen below is an example of basic variables been created and used to print out information.

# Holds a String carName = "Honda Civic" # Holds an Integer (or a whole number) carMaxSpeed = 120 # Holds a list of things carList = ["Honda Civic", "Chevy Colorado", "Ford Mustang"] # A sentence with the variable added at the end. print("The car is a " + carName) # A sentence showcasing the carMaxSpeed variable print("The maximum speed of the car is: " + carMaxSpeed) # Printing out the first item in the list. print(carList[0])
If Statements

If statements are statements that first check to see if a condition is true and then perform specific functionality if it is true.

This can be used to perform various functionality that requires a condition to be met. The syntax is as followed.

if numberBeingChecked == otherNumber: print("The number being checked is the same as the otherNumber!")

In a number guessing game you could check to see if the correct number was guessed and print out a congratulatory statement.

# The magic number winningNumber = 7 # Store the number the user typed guessedNumber = input("Enter a number between 1 and 10") if(guessedNumber == winningNumber): print("Congratulations you guessed the magic number!") # Else performs the following code if the above statement is not true. else: print("Sorry! That is not the magic number.")
While Loops

While loops are a type of loop that repeats the code contained inside until a certain condition is met that stops it from looping.

The syntax for a while loop is as follows:

while(condition < maxLoopAmount): print("This text will print repeatedly until the condition is met.")

You can use a while loop in the previous guessing game we created to keep looping until the player correctly guesses the magic number.

# The magic number winningNumber = 7 # Store the number the user typed guessedNumber = input("Enter a number between 1 and 10") # A boolean variable that holds either a true or false value # In this case a false variable until the player wins the game playerWin = False # Will continue looping until playerWin is set to True while(playerWin == False): if(guessedNumber == winningNumber): print("Congratulations you guessed the magic number!") # Set playerWin to true now since the player won so the game will end playerWin = True # Else performs the following code if the above statement is not true. else: print("Sorry! That is not the magic number.")
Functions

Functions are entire chunks of code that you can contain inside what's known as a function so that you can reuse them without having to write the same code.

A function has the following syntax:

def my_new_function(): print("The new function has been activated!") my_new_function()

The def is short for define and is required when creation a function. The parenthesis () can be used to send information to the function if needed. In this case it is empty because we aren't sending it any information. We're simply running the code inside the function any time it's called. Underneath the function is the syntax for calling the function so that you can use it. You simply need the name and the parenthesis. If the parenthesis requires information to be sent you must include that too. Below is an example of the parenthesis containing variables for sending information.

def my_new_function(printThisToo): print("The new function has been activated!") print(printThisToo) my_new_function("I like pie")

Adding the variable printThisToo will hold whatever information is placed in the parenthesis when the function is called.
